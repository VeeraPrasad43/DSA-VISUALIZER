🚀 DSA Practice Website
This is a simple Data Structures and Algorithms (DSA) practice website built to help students and developers improve their problem-solving skills.
The website organizes DSA problems in a structured way so users can easily learn and practice topic by topic.
📌 Features
Topic-wise DSA problems
Covers Arrays, Strings, Linked List, Stack, Queue, Trees, Graphs, etc.
Beginner-friendly structure
Simple and clean UI
Easy to navigate and practice
🛠️ Tech Stack
HTML
CSS
JavaScript
🚀 How to Run
Clone the repository
git clone https://github.com/VeeraPrasad43/DSA-VISUALIZER
Open the project folder
Run index.html in your browser
